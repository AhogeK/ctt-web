description: 主动记忆 - 保存项目状态和变更到记忆库（防止 AI 幻觉遗忘）
---

**⚠️ 强制规则**: 此命令必须严格按照 AGENTS.md 执行，作为自动记忆的补充手段。

## 执行流程

### 阶段 1: 读取现有记忆文件

读取 `memory-bank/` 目录下的所有文件，确认现有状态。

### 阶段 2: 识别并更新

根据 AGENTS.md R2 的更新映射表，识别需要记录的新信息并更新对应文件。

**时间获取**: 使用 `date "+%Y-%m-%d"` 获取当前日期

### 阶段 3: 输出摘要

```
=== 记忆保存 ===

已更新:
- [文件]: [更新内容摘要]

保留未更新:
- [文件]: [原因]
```

## 触发场景

- 完成重要任务 / 做出关键决策
- 遇到并解决问题（如 Hydration mismatch、类型错误、构建问题）
- 切换 feature 上下文 / 长时间对话
- 用户明确要求保存

## 禁止事项

- **不要假设** - 必须先读取再更新
- **不要遗漏日期** - 必须包含 YYYY-MM-DD
- **不要冗余** - 只记录新信息
